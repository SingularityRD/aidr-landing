# Singularity AIDR — Responsible Disclosure Policy
#
# We commit to:
#  - Acknowledge receipt within 3 business days
#  - Triage within 10 business days
#  - Coordinate disclosure timeline (90 days default)
#  - Credit researchers in our Hall of Fame (with permission)
#  - Pay bounty awards for valid critical/high findings
#
# Scope:
#  - aidr.dev, app.aidr.dev, api.aidr.dev
#  - AIDR CLI binary (aidr.dev/install.sh)
#  - AIDR Scan Docker images (ghcr.io/singularityrd/aidr-scan)
#  - AIDR SDK packages (@aidr/sdk, aidr-sdk, github.com/singularityrd/aidr-sdk)
#
# Out of scope:
#  - Vulnerabilities in third-party dependencies (please report upstream)
#  - Denial-of-service attacks
#  - Social engineering of SingularityRD staff
#  - Physical security
